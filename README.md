# unsupervised-music-generation
Unsupervised Music Generation
Project Overview:

This project focuses on generating music automatically using deep learning techniques without labeled data.
We use MIDI datasets and train multiple models to learn musical patterns and generate new compositions.

Models Used: 
LSTM Autoencoder
Variational Autoencoder (VAE)
Transformer
Reinforcement Learning (RLHF) for fine-tuning
Baseline models (Random, Markov Chain)

How to Run:
1) Open the notebook
2) Run all cells step by step

Evaluation Metrics

Evaluate the  generated music using:

Pitch Histogram Distance
Rhythm Diversity
Repetition Ratio
Human Evaluation Score

Notes:
Dataset used: MAESTRO MIDI Dataset
Project developed using Google Colab


Run the notebook directly in Google Colab for full execution.
